# src\http\api.ts:
- apiGetUsers: Obtém a lista completa de integrantes.
- apiGetUser: Obtém informações detalhadas de um usuário específico.
- apiLogin: Realiza a autenticação JWT do usuário.
- apiRegister: Cria uma nova conta de usuário.
- apiCreateWallet: Ativa a carteira de EcoCoins do usuário.
- apiDeleteUser: Remove o usuário do banco.
- apiUpdateUser: Atualiza os dados cadastrais do usuário.
- apiTransaction: Executa transações na carteira (Crédito, Débito e Resgate).
- apiGetWalletBalance / apiGetWallet: Sincroniza e confirma saldos de carteiras.
